# NroOffline

Game RPG Offline 2D được phát triển bằng Unity và C#.

## Giới thiệu

NroOffline là project cá nhân được phát triển nhằm học tập và thực hành các hệ thống gameplay phổ biến trong game RPG/MMORPG.

Project tập trung vào xây dựng:
- Hệ thống chiến đấu
- Kỹ năng
- AI quái
- Inventory
- Shop NPC
- UI gameplay
- Animation và quản lý trạng thái nhân vật

## Tính năng hiện có

- Di chuyển nhân vật
- Hệ thống chiến đấu
- Kỹ năng và animation
- AI quái vật (đuổi và tấn công)
- Chọn mục tiêu enemy
- Hiển thị thông tin enemy
- Hệ thống Inventory
- Hệ thống Shop NPC
- Hồi sinh nhân vật
- Quản lý Scene
- Hệ thống UI tương tác

## Công nghệ sử dụng

- Unity Engine
- C#
- Unity Animator
- Unity UI System
- Git & GitHub
